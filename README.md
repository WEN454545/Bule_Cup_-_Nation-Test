# 蓝桥杯十六届国赛项目

## 项目简介
本项目是蓝桥杯十六届国赛的参赛作品，基于STC15F2K60S2单片机开发，实现了一个多传感器数据采集与监控系统。系统能够实时监测温度、液位、重量等参数，并通过数码管显示，同时具备串口通信、异常检测和报警功能。

## 功能特点
- **多传感器数据采集**：
  - 温度传感器（DS18B20）
  - 超声波测距传感器（用于液位检测）
  - 重量传感器（通过ADC采集）
  - NE555频率检测
- **数据显示**：
  - 8位数码管动态显示
  - 多种显示模式：时间、液位、体积、重量
  - 参数设置界面
  - 传感器标定界面
- **串口通信**：
  - 9600bps波特率
  - 支持参数设置和查询
  - 异常状态上报
- **异常检测**：
  - 水位异常（过高/过低）
  - 频率异常（超值、连续变化、变化过大）
  - LED指示灯和蜂鸣器报警
- **按键控制**：
  - 模式切换
  - 参数调整
  - 传感器标定

## 硬件架构
- **主控芯片**：STC15F2K60S2
- **显示模块**：8位共阳极数码管
- **输入模块**：4x4矩阵键盘
- **输出模块**：8位LED指示灯、蜂鸣器
- **传感器**：
  - DS18B20温度传感器
  - 超声波测距模块
  - 重量传感器（通过ADC）
  - NE555频率模块
- **通信模块**：串口通信

## 使用说明

### 编译环境
- Keil C51 v9.56
- STC-ISP下载工具

### 硬件连接
- 按照赛题要求连接各个模块
- 确保传感器正确接线
- 连接串口到计算机

### 操作流程
1. **启动系统**：系统上电后自动初始化，显示时间
2. **模式切换**：按S4键切换显示模式
   - 模式0：数据显示（时间、液位、体积、重量）
   - 模式1：参数设置
   - 模式2：传感器标定
3. **参数设置**：在模式1下，按S5键切换参数，按S8/S9键调整参数值
4. **传感器标定**：在模式2下，按S5键切换标定项，按S8/S9键调整标定值
5. **串口通信**：通过串口发送命令设置或查询参数

### 串口命令格式

#### 参数设置
(H1,1.5)(H2,0.3)(F,2000)(S,2)(r,1.0)(L,1.0)(W,1.0)(H,1.0)
- H1：液面上限（单位：米）
- H2：液面下限（单位：米）
- F：频率参数
- S：容器类型（0：球体，1：圆柱体，2：长方体）
- r：半径参数（单位：米）
- L：长度参数（单位：米）
- W：宽度参数（单位：米）
- H：高度参数（单位：米）

#### 参数查询
(H1?)(H2?)(F?)(S?)(r?)(L?)(W?)(H?)

#### 时间设置
(T:235950)
设置时间为23:59:50

#### 异常上报格式
- 水位过高：`[500,HHMMSS,XX.XX]`
- 水位过低：`[501,HHMMSS,XX.XX]`
- 频率超值：`[503,HHMMSS,XXXX]`  **与led的联动存在问题**
- 频率连续变化：`[504,HHMMSS,X]` **无法正常产生异常信号与led无法联动**
- 频率变化过大：`[504,HHMMSS,X]` **无法正常产生异常信号与led无法联动**
